# Feature Request — Governance Status Helper

## Goal
Voeg een klein, read-only helper script toe dat direct laat zien welke governance-bronnen canoniek zijn in deze repo.

## Waarom
Ik wil sneller kunnen zien:
- waar de CTO rules staan
- waar het universal audit-proces staat
- wat de primary quality gate is
- welke skill-bron canoniek is

## Constraints
- Geen massive refactor
- Alleen kleine, gerichte wijziging
- Moet passen binnen repo-10x guardrails
- Moet verifieerbaar zijn met bestaande quality gates

# Memory Diagnostics Command

Diagnose Claude Code memory leaks and context loss issues.

**Use when:**
- Claude suddenly "forgets" project details
- CLAUDE.md files seem to be ignored
- MCP servers disconnect or become unresponsive
- Context from previous messages is lost
- Claude acts confused about project structure

**This command will:**
1. Check if CLAUDE.md files are being loaded properly
2. Verify MCP server connections and status
3. Analyze memory cache integrity
4. Test context retention capabilities
5. Generate detailed diagnostics report
6. Suggest specific fixes for detected issues

**Arguments:** $ARGUMENTS (optional: 'verbose' for detailed output)

**Output:** Comprehensive memory diagnostics and repair suggestions

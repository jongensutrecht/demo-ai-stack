# E2E (Short Alias)

Shortcut voor `/e2e-validate`. Zie die command voor volledige documentatie.

**Laad de volledige skill:**

Voer uit: Lees en volg `C:\Users\Alexander\.claude\commands\e2e-validate.md`

**Quick reference:**
```
/e2e                    → Volledige project validatie
/e2e {feature}          → Test specifieke feature
/e2e {url}              → Test specifieke URL/endpoint
/e2e --api-only         → Alleen API tests
/e2e --ui-only          → Alleen UI tests (Playwright)
```

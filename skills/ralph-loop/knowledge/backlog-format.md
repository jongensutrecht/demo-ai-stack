# BACKLOG.md Format

## Locatie

`stories_claude/<Process>/BACKLOG.md`

## Template

```markdown
# BACKLOG - <Process>

| Story | Status | Attempts | Notes |
|-------|--------|----------|-------|
| OPS-001 | [TODO] | 0 | |
```

## Statussen

| Status | Betekenis |
|--------|-----------|
| `[TODO]` | Nog niet gestart |
| `[IN_PROGRESS]` | Bezig |
| `[DONE]` | Compleet |
| `[BLOCKED]` | 3x gefaald |

## Details

Details en voorbeelden: `python scripts/backlog.py`
